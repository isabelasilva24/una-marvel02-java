# 🧠 Projeto: Marvel Switch (Java)

## 📌 Descrição

Este projeto em Java demonstra o uso da estrutura `switch`, comparando:

* Switch tradicional
* Switch moderno (Switch Expression)

O sistema retorna o status da personagem Wanda Maximoff conforme sua fase.

---

## 🚀 Objetivo

* Praticar estruturas condicionais em Java
* Entender a evolução do `switch`
* Melhorar a legibilidade do código

---

## 🛠️ Tecnologias

* Java (JDK 14+)

---
